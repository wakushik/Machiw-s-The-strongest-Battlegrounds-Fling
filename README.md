## [ About Script ]
My tool developed specifically for "The Strongest Battlegrounds", you can use it in other games but it can be buggy or even not work.

## [ Features ]

*  Teleport System
  - Teleport to player all the time if he in fling radius, if he is far enough you not gonna be teleported.

* Rejoin Fix 
  - Fixed most of all loop fling problem, rejoin bug. So now if player rejoin, it still will be flinged.

* Settings
  - You can configure power of fling and attack time (How much time player is trying to fling).
